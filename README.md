# 基于 LangGraph 诸葛亮三国人物问答 Agent 开发文档

## 一、项目简介

本项目基于 LangGraph搭建多分支流程 Agent，以诸葛亮为角色主体，内置《三国演义》全有名有姓人物关系库，实现意图识别、闲聊、人物咨询、情绪安抚多分支逻辑。

整体采用纯内存数据模拟图关系，无需额外部署图数据库，代码结构规范、可直接运行，支持键盘交互式问答。

### 核心能力

1. 自动识别用户意图：闲聊、人物咨询、负面情绪 / 投诉三类
2. 基于内置三国人物关系库，以诸葛亮古风口吻回答人物相关问题
3. 针对负面情绪做安抚处理，完善对话闭环
4. 采用 `TypedDict` 规范状态结构，LangGraph 链式语法构建工作流

## 二、技术栈

| 技术 / 库  |               作用               |
| :--------: | :------------------------------: |
| LangGraph  | 构建多节点、多分支 Agent 工作流  |
| LangChain  |      调用大模型、封装消息体      |
| ChatOpenAI | 对话大模型（生成回答、风格润色） |
| TypedDict  |     规范 Agent 全局状态结构      |
|   Python   |           主力开发语言           |

## 三、整体架构说明

### 3.1 工作流流程图

```
START 开始
    ↓
意图识别节点
    ├─ 闲聊意图 → 闲聊节点 → END 结束
    ├─ 咨询意图 → 咨询节点 → END 结束
    └─ 投诉/负面情绪 → 投诉情感节点 → 安抚节点 → 判定结束 → END
```



### 3.2 状态结构（TypedDict）

统一管理 Agent 运行时所有状态，代码强规范、易维护：

```
class State(TypedDict):
    messages: list        # 对话历史消息列表
    intent: str          # 识别出的用户意图：闲聊/咨询/投诉
    complaint_resolved: bool  # 投诉情绪是否已安抚完成
```



### 3.3 节点功能划分

1. **意图识别节点**：解析用户输入文本，判断当前对话意图
2. **闲聊节点**：处理日常闲聊对话，调用大模型以诸葛亮口吻回应
3. **咨询节点**：核心节点，查询三国人物关系并作答
4. **投诉情感节点**：识别负面情绪，标记情绪状态
5. **安抚节点**：针对负面情绪进行安抚，标记情绪已处理

### 3.4 路由规则

- 意图路由：根据识别结果分发至「闲聊 / 咨询 / 投诉」对应节点
- 投诉路由：安抚完成后直接结束对话

## 四、核心数据说明

### 4.1 人物关系库

使用字典模拟图数据库节点与关系，key为三国人物姓名，value为诸葛亮视角的人物关系、身份、事迹描述。



 数据覆盖范围：

- 蜀汉集团：主公、五虎将、文臣、武将、后辈、南中势力
- 曹魏集团：曹氏宗族、五子良将、谋臣、名将、后期主力
- 东吴集团：孙家三代、四大都督、十二虎臣、文臣名士
- 群雄势力：董卓、吕布、袁绍、刘表、刘璋、黄巾起义军等
- 诸葛亮亲友 & 荆州隐士圈层：水镜先生、徐庶、岳父、同门好友等

### 4.2 角色人设 Prompt

固定角色设定，约束大模型输出风格：

- 身份：蜀汉丞相 诸葛亮（字孔明，号卧龙）
- 语气：儒雅沉稳、智虑深远、古风文言
- 规则：严格依据内置人物关系作答

## 五、运行说明

### 5.1 交互规则

1. 正常提问人物：如 `司马懿是谁`、`孟获和你是什么关系` → 进入**咨询分支**，正常作答
2. 日常打招呼：如 `你好`、`在吗` → 进入**闲聊分支**
3. 输入负面词汇：如 `不满`、`生气` → 进入**投诉分支**，先安抚再结束
4. 输入 `exit` 退出整个对话程序

### 5.2 运行示例

你：司马懿是谁？
⚪ 孔明：司马懿乃我的毕生劲敌，其人隐忍多智，数次与我两军对垒，极难对付。
你：孟获与你有何渊源？
⚪ 孔明：孟获为南中首领，我率军南征，七擒七纵，终令其心悦诚服，南疆自此安定。
你：你好呀
⚪ 孔明：阁下有礼，不知今日前来，想问些什么？
你：有点生气
⚪ 孔明：请勿动怒，在下愿为你一一解惑。
你：exit
孔明：告辞，再会。
